Installing Wok

First of all, download Wok package according to your operating system.

We provide packages for Fedora 25, openSUSE LEAP 42.2, Ubuntu 17.04 and CentOS 7. Those packages were built on and for x86_64 machines.

After it, install Wok package using the default packaging management tool in your system. You can do it by command line like below:

Fedora and RHEL:

sudo yum install <wok.rpm>

Ubuntu:

sudo dpkg -i <wok.deb>

If your system does not have all the dependencies installed, the above command will fail. To fix it and install the dependencies needed, run:

sudo apt-get install -f

openSUSE:

sudo zypper install <wok.rpm>

Install Wok Plugins

Wok is a web server framework, so you might want to install some plugins to see it on action.

Ginger: Wok plugin for host managment

Ginger Base: Basic host management for WoK

Kimchi: An HTML5 management interface for KVM

Starting wokd service

After installing wok package, you just need to start its service to get it up:

sudo service wokd start

Note: The above command line may vary depends on your operating system.

Then access Wok in your favorite browser https://<machine-ip>/8001

Have fun! =)
